Contacting IT

Send a message to it-help@cagreens.org to submit a request for services or technical help. But first, read the IT FAQ

The IT FAQ (frequently asked questions) answers many common questions and provides guidance for using IT resources.

Hosting, Email Addresses, & User Accounts

If you would like to host a web site for an entity of the Green Party, or you would like a cagreens.org email address, you must:

  • Have a cagreens "users account" (see User Accounts below); and
  • Have authorization from your County Council, committee coordinator, or other such responsible party.

User Accounts

If you need a user account on the cagreens.org network, please fill out the application form and read the terms of agreement.

Lists

Email Lists

A list of Green Party email lists to which you can subscribe is at http://lists.cagreens.org/mailman/listinfo/. These are the lists hosted on Mailman. There are a few other lists on Majordomo that are not included here. Our goal is to move all lists to Mailman. Also, there are a few private lists that are by invitation only and are not listed here.

Broadcast Lists

View the rules and procedures that govern the operation of our broadcast lists, which are the County Contact list and the Updates list. Read these tips on formating your messages for the broadcast lists.

General Discussion List

Common Password

The GPCA web site has several "internal" pages that require a user ID and password. This is a common ID and password used on all internal pages. This password is changed anually after the first General Assembly of each year.

The ID and password is generally available to all active Greens. This policy provides the details.
